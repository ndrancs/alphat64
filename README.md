alphat os
=========

####Lightweight, portable and flexible operating system

alphat os is a simple and minimalistic Linux distribution for the i686, x86-64 architecture, built using SPKG set of scripts developed by Simargl. It is based on Arch Linux and uses pacman as the default package manager. This operating system features highly configurable and lightweight Openbox window manager. Modular by design, alphat os makes it easy to add desired functionality. 

This repository contains all bash scripts used for making alphat os Linux based live CDs, and also set of several home-grown applications written in Vala and GTK+3: Emendo - text editor with syntax highlighting, Simple Radio - play online radio stations from the system tray, Taeni - terminal emulator, and others located in def-scripts/03_extra directory.

This os mainly use init from fatdog64 v630.
